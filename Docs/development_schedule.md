## 리스크 및 주의사항

### 고위험 태스크

1. **오디오/애니메이션 시스템 (Week 3-4)**
   - 에셋 품질이 게임 필에 직접적 영향
   - 대응: 플레이스홀더로 먼저 구현, 이후 에셋 교체

2. **벌목꾼 군중 시스템 (Week 8)**
   - 다수 캐릭터 동시 처리로 성능 이슈 발생 가능
   - 대응: 조기에 프로파일링, 오브젝트 풀링 우선 적용

3. **통합 시스템 (Task 8.3)**
   - 시각적 통합과 내부 로직 분리 복잡
   - 대응: 명확한 데이터 구조 설계 선행

### 크리티컬 패스

```
M1 → 3.1 → 4.1 → 6.1 → 8.1 → 8.3
(프로토타입) → (오디오) → (벌목꾼 애니메이션) → (크리티컬) → (벌목꾼 5종) → (통합 시스템)
```

이 경로의 지연은 전체 일정에 직접 영향을 미침.

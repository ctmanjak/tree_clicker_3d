## 1. 게임 개요

### 1.1 컨셉

'벌목왕'은 나무를 베어 목재를 수집하고, 업그레이드를 통해 점점 더 강력한 벌목꾼으로 성장하는 클리커/방치형 게임입니다. 플레이어는 간단한 클릭 조작으로 시작하여 자동화 시스템을 구축하고, 궁극적으로 신비로운 숲의 전설적인 나무까지 정복하는 것을 목표로 합니다.

### 1.2 기본 정보

| 항목 | 내용 |
|------|------|
| 장르 | 클리커 / 방치형 (Idle/Incremental) |
| 플랫폼 | 모바일 (iOS, Android), 웹 브라우저 |
| 타겟 유저 | 캐주얼 게이머, 방치형 게임 팬 (10대~30대) |
| 플레이 시간 | 세션당 5~30분, 장기 플레이 가능 |
| 수익 모델 | 광고 시청 보상, 인앱 구매 (프리미엄 도끼, 부스터) |

### 1.3 핵심 재미 요소

- **즉각적인 피드백:** 클릭할 때마다 나무 조각이 튀고, 목재 수치가 올라가는 시각적 쾌감
- **성장의 만족감:** 묘목에서 거대한 나무로, 나무 도끼에서 황금 도끼로 눈에 보이는 발전
- **자동화의 즐거움:** 벌목꾼들을 고용해 우르르 몰려다니며 자동으로 목재를 수집하는 모습 관찰
- **수집 욕구 충족:** 다양한 도끼와 배경을 해금하며 컬렉션 완성

## 2. 핵심 게임 메카닉

### 2.1 기본 조작

| 조작 | 동작 | 결과 |
|------|------|------|
| 탭/클릭 | 화면 중앙의 나무 터치 | 목재 획득 (기본 1개) |
| 스와이프 | 업그레이드 패널 열기/닫기 | UI 전환 |

### 2.2 핵심 루프

게임의 핵심 게임플레이 루프는 다음과 같이 구성됩니다:

1. **클릭** → 나무를 클릭하여 목재 획득
2. **수집** → 목재가 상단 카운터에 누적
3. **투자** → 목재로 업그레이드 구매
4. **강화** → 클릭당 획득량 또는 자동 획득량 증가
5. **진화** → 일정 목표 달성 시 나무/배경/도끼 변화
6. **반복** → 더 높은 목표를 향해 루프 반복

### 2.3 크리티컬 시스템

클릭 시 일정 확률로 크리티컬 히트가 발생하여 추가 보상을 획득합니다.

| 항목 | 기본값 | 업그레이드 시 |
|------|--------|---------------|
| 발동 확률 | 5% | 최대 25% |
| 배율 | 2배 | 최대 5배 |
| 시각 효과 | 번개 이펙트 | 황금 번개 + 화면 흔들림 |
| 사운드 | 타격음 강조 | 특수 효과음 추가 |

## 3. 경제 및 진행 시스템

### 3.1 재화 시스템

| 재화 | 획득 방법 | 용도 |
|------|-----------|------|
| 목재 (기본) | 나무 클릭, 자동 수집, 광고 보상 | 업그레이드 구매, 벌목꾼 고용 |
| 황금 목재 (프리미엄) | 업적 달성, 일일 보너스, 인앱 구매 | 특수 업그레이드, 즉시 부스트 |
| 잎사귀 (이벤트) | 시즌 이벤트, 특별 미션 | 한정 아이템 교환 |

### 3.2 밸런싱 공식

업그레이드 비용과 효과는 다음 공식을 따릅니다:

- **비용 증가:** 기본 비용 × 1.15^(현재 레벨)
- **효과 증가:** 기본 효과 × (1 + 0.1 × 현재 레벨)
- **투자 회수:** 각 업그레이드는 평균 2~3분 내 투자금 회수 설계

### 3.3 마일스톤

| 목재 수량 | 해금 콘텐츠 |
|-----------|-------------|
| 100 | 날카로운 도끼 구매 가능 |
| 500 | 나무 → 일반 나무로 성장 |
| 1,000 | 숙련 벌목꾼 해금 |
| 5,000 | 배경 변경 (숲 → 정글) |
| 10,000 | 전설의 도끼 해금 |
| 50,000 | 신비로운 숲 배경 해금 |
| 100,000 | 숲의 정령 최종 해금 |

## 4. 업그레이드 시스템

### 4.1 클릭 업그레이드

클릭당 획득하는 목재량을 증가시키는 업그레이드입니다.

| 이름 | 비용 | 효과 | 설명 |
|------|------|------|------|
| 날카로운 도끼 | 목재 100 | +1/클릭 | 잘 갈린 날로 효율 상승 |
| 강철 도끼 | 목재 500 | +5/클릭 | 단단한 강철로 제작된 도끼 |
| 전설의 도끼 | 목재 2,000 | +20/클릭 | 고대 장인의 걸작 |
| 미스릴 도끼 | 목재 10,000 | +100/클릭 | 마법이 깃든 신비로운 도끼 |
| 용암 도끼 | 목재 50,000 | +500/클릭 | 용의 숨결로 단조된 도끼 |

### 4.2 벌목꾼 소환 업그레이드

벌목꾼 캐릭터를 소환하여 자동으로 나무를 베게 합니다. 소환된 벌목꾼은 화면에 등장하여 직접 나무를 타격하는 애니메이션을 보여줍니다.

| 이름 | 비용 | 효과 | 외형/특징 |
|------|------|------|-----------|
| 견습 벌목꾼 | 목재 200 | +1/초 | 작은 체구, 나무 도끼, 느린 타격 |
| 숙련 벌목꾼 | 목재 1,000 | +10/초 | 건장한 체격, 강철 도끼, 빠른 타격 |
| 대장 벌목꾼 | 목재 5,000 | +50/초 | 거구의 근육질, 쌍도끼, 강력한 타격 |
| 드워프 장인 | 목재 25,000 | +250/초 | 작지만 강한 드워프, 룬 도끼 |
| 숲의 정령 | 목재 100,000 | +1,000/초 | 반투명한 정령, 마법 공격 |

#### 벌목꾼 군중 시스템

벌목꾼을 구매할 때마다 새로운 캐릭터가 화면에 등장하여 우르르 몰려다니며 나무를 벱니다. 성능 최적화를 위해 최대 인원 제한과 시각적 통합 시스템이 적용됩니다.

- **개별 소환:** 업그레이드 구매 시 해당 벌목꾼 1명이 화면에 추가 등장
- **군중 행동:** 벌목꾼들이 나무 주변을 자유롭게 돌아다니며 타격
- **타격 타이밍:** 각자 독립적인 타이밍으로 도끼질, 리드미컬한 효과

#### 인원 제한 및 시각적 통합

성능 최적화를 위해 화면에 표시되는 벌목꾼 수를 제한합니다. 제한을 초과하면 같은 종류의 벌목꾼이 시각적으로 통합되어 1명으로 표시되지만, 내부적으로는 여러 명의 효과가 그대로 적용됩니다.

| 항목 | 수치 | 설명 |
|------|------|------|
| 최대 동시 표시 | 20명 | 화면에 동시에 표시되는 벌목꾼 수 제한 |
| 통합 발동 조건 | 20명 초과 시 | 같은 종류 벌목꾼 중 가장 많은 종류부터 통합 |
| 통합 비율 | 5명 → 1명 | 5명이 1명으로 표시됨 (효과는 5명분 그대로) |
| 통합 벌목꾼 효과 | 정확히 5명분 | 보너스 없이 5명의 효과를 그대로 적용 |

#### 통합 벌목꾼 시각 표현

통합된 벌목꾼은 '여러 명을 대표한다'는 것을 플레이어가 인지할 수 있도록 표시합니다.

- **인원 표시:** 머리 위에 'x5', 'x10' 등 대표 인원 수 표시
- **타격 속도:** 5명분을 표현하기 위해 타격 애니메이션 속도 증가
- **파티클 증가:** 타격 시 파티클이 더 많이 발생 (5명이 때리는 느낌)

### 4.3 특수 업그레이드

| 이름 | 비용 (황금 목재) | 효과 |
|------|------------------|------|
| 행운의 부적 | 10 | 크리티컬 확률 +5% |
| 황금 장갑 | 25 | 크리티컬 배율 +1배 |
| 시간의 모래시계 | 50 | 자동 수집 속도 2배 (30분) |
| 무한의 숲 | 100 | 오프라인 수집량 2배 |

## 6. 게임 필 (Game Feel)

### 6.1 타격감 구현

클릭의 만족감을 극대화하기 위한 다층적 피드백 시스템을 구현합니다.

- **화면 흔들림:** 클릭 시 0.05초간 미세한 화면 진동 (강도: 2~5px)
- **나무 흔들림:** 클릭 방향 반대로 15도 기울어졌다 복귀 (0.2초)
- **파티클 효과:** 나무 조각 3~5개 + 잎사귀 2~3개 튀어나감
- **플로팅 텍스트:** 획득 목재량이 클릭 위치에서 위로 떠오르며 페이드아웃

### 6.2 크리티컬 히트 연출

- **번개 이펙트:** 화면 중앙에서 번개가 치는 효과 (0.3초)
- **화면 플래시:** 황금색 플래시 효과
- **확대 텍스트:** 'CRITICAL!' 텍스트가 크게 표시 후 축소
- **추가 파티클:** 황금빛 입자가 폭발하듯 퍼짐

### 6.3 업그레이드 구매 연출

- **버튼 애니메이션:** 구매 가능 시 살짝 커졌다 작아지는 펄스 효과
- **구매 확정:** '짠!' 효과음 + 버튼에서 별 파티클 방출
- **레벨업 표시:** 업그레이드 아이콘 위에 레벨 숫자 +1 애니메이션

### 6.4 벌목꾼 군중 연출

- **소환 애니메이션:** 구매 시 화면 밖에서 뛰어 들어오며 등장
- **군중 이동:** 나무 주변을 우르르 돌아다니며 자연스러운 무리 행동
- **개별 AI:** 각 벌목꾼이 독립적으로 이동, 정지, 타격 반복
- **충돌 회피:** 서로 겹치지 않게 자연스럽게 비켜가는 움직임
- **타격 타이밍:** 제각각 다른 타이밍에 도끼질 → 통통통 리드미컬한 소리

#### 통합 연출

인원 초과 시 통합은 자연스럽게 이루어지며, 플레이어의 게임 흐름을 방해하지 않습니다.

- **통합 과정:** 같은 종류 벌목꾼들이 자연스럽게 한 곳으로 모여듦
- **페이드 처리:** 4명이 서서히 사라지고 1명만 남음
- **인원 표시 등장:** 남은 1명 머리 위에 'x5' 숫자가 팝업
- **간결한 알림:** 화면 하단에 '벌목꾼 통합: 견습 벌목꾼 x5' 작은 토스트 메시지

### 6.5 진화 연출

- **화면 전환:** 밝은 빛과 함께 새로운 스테이지로 전환 (1.5초)
- **축하 메시지:** '나무가 성장했습니다!' 배너 표시
- **BGM 변화:** 새 스테이지에 맞는 BGM으로 크로스페이드

## 7. UI/UX 디자인

### 7.1 화면 구성

메인 게임 화면은 직관적이고 깔끔한 레이아웃으로 구성됩니다.

| 영역 | 위치 | 구성 요소 |
|------|------|-----------|
| 헤더 | 상단 | 목재 카운터, 황금 목재, 설정 버튼 |
| 메인 | 중앙 | 나무 (터치 영역), 도끼 애니메이션 |
| 벌목꾼 영역 | 나무 주변 | 우르르 돌아다니는 벌목꾼들 (최대 20명) |
| 진행 바 | 중앙 하단 | 다음 나무 성장 단계까지의 진행도 |
| 업그레이드 | 우측 | 세로 스크롤 업그레이드 목록 |
| 퀵 메뉴 | 좌측 하단 | 업적, 컬렉션, 상점 버튼 |

### 7.2 목재 카운터

- **위치:** 화면 상단 중앙
- **디자인:** 나무 프레임 + 목재 아이콘 + 숫자
- **숫자 표기:** 1,000 이상은 'K', 1,000,000 이상은 'M' 표기
- **색상 변화:** 흰색(기본) → 노란색(1K+) → 주황색(10K+) → 금색(100K+)

### 7.3 업그레이드 패널

- **기본 상태:** 우측에 접힌 형태로 아이콘만 표시
- **확장 시:** 좌측으로 슬라이드하며 전체 목록 표시
- **구매 가능 표시:** 테두리 반짝임 + 알림 뱃지
- **구매 불가 표시:** 흑백 처리 + 필요 목재량 표시

### 7.4 진행도 바

- **디자인:** 나무 껍질 텍스처의 막대
- **채워짐:** 녹색 잎사귀 패턴으로 채워짐
- **목표 표시:** 바 끝에 다음 진화 단계 아이콘

## 10. 기술 사양

### 10.1 개발 환경

| 항목 | 스펙 |
|------|------|
| 게임 엔진 | Unity 2022.3 LTS / Godot 4.x |
| 프로그래밍 언어 | C# (Unity) / GDScript (Godot) |
| 타겟 해상도 | 1080×1920 (9:16 세로) |
| 프레임 레이트 | 60 FPS 목표 |
| 최소 사양 | iOS 12+ / Android 7.0+ |

### 10.2 데이터 저장

- **로컬 저장:** PlayerPrefs / JSON 파일
- **클라우드 저장:** Google Play Games / Game Center 연동
- **저장 데이터:** 목재량, 업그레이드 레벨, 해금 상태, 통계
- **자동 저장:** 30초마다 + 앱 백그라운드 전환 시

### 10.3 벌목꾼 군중 최적화

다수의 벌목꾼 캐릭터가 동시에 움직이는 시스템을 위한 최적화 전략입니다.

- **오브젝트 풀링:** 벌목꾼 오브젝트를 미리 생성해두고 재활용
- **LOD 시스템:** 화면 가장자리 벌목꾼은 간소화된 애니메이션 적용
- **배칭 렌더링:** 같은 종류 벌목꾼은 하나의 드로우 콜로 처리
- **AI 업데이트 분산:** 모든 벌목꾼을 매 프레임 업데이트하지 않고 분산 처리
- **시각적 통합:** 20명 초과 시 자동으로 통합 표시하여 오브젝트 수 관리

### 10.4 오프라인 시스템

앱을 종료해도 자동 수집이 계속되어 다시 접속 시 보상을 받습니다.

- **최대 오프라인 시간:** 24시간
- **오프라인 효율:** 온라인 대비 50% (업그레이드로 증가 가능)
- **광고 보너스:** 광고 시청 시 오프라인 보상 2배

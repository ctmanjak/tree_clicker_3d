## 밸런싱 가이드

### 초기 밸런스 (M1)

| 항목 | 값 | 비고 |
|------|-----|------|
| 기본 클릭 획득량 | 1 | |
| 날카로운 도끼 기본 비용 | 100 | |
| 날카로운 도끼 비용 배율 | 1.15 | |
| 날카로운 도끼 효과 | +1/클릭 | |
| 벌목꾼 수집량 | 1/초 | |

### 밸런스 조정 포인트

1. **클릭 반응성:**
   - 너무 느리면 지루함
   - 너무 빠르면 RSI 위험

2. **업그레이드 진행:**
   - 첫 업그레이드까지 100클릭 (약 30초)
   - 적절한 성취감 필요

3. **벌목꾼 가치:**
   - 1/초 = 3600/시간
   - 수동 클릭 대비 효율성 체크
